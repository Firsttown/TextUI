# Примеры использования фреймворка TextUI

В этой папке содержится пример использования фреймворка TextUI, демонстрирующий его основные возможности.

## Содержание

- [main.cpp](main.cpp) - Основной файл примера
- [CMakeLists.txt](CMakeLists.txt) - Файл для сборки проекта с помощью CMake

## Возможности, продемонстрированные в примере

### 1. Основные виджеты
- Создание окон с заголовками и рамками
- Добавление меток (Label)
- Текстовые поля для ввода (TextBox)
- Чекбоксы (CheckBox)
- Прогресс-бары (ProgressBar)
- Кнопки с обработчиками событий

### 2. Модальные окна
- Информационные окна (MessageBox)
- Окна подтверждения (ConfirmDialog)
- Диалоги ввода данных (InputDialog)
- Вложенные модальные окна

### 3. Обработка событий
- Обработка нажатий кнопок
- Получение данных из текстовых полей
- Работа с состояниями чекбоксов
- Обработка результатов диалоговых окон

## Сборка и запуск

Для сборки примера используйте CMake:

```bash
mkdir build
cd build
cmake ..
make
```

После сборки запустите исполняемый файл:

```bash
./TextUI_Example
```

В Windows:

```bash
TextUI_Example.exe
```

## Структура примера

Пример состоит из двух основных демонстраций:

1. **Основные виджеты** - демонстрирует работу с различными виджетами:
   - Текстовые поля для ввода
   - Чекбоксы
   - Прогресс-бары
   - Кнопки с обработчиками

2. **Модальные окна** - демонстрирует работу с модальными диалогами:
   - Информационные окна
   - Окна подтверждения
   - Диалоги ввода данных
   - Вложенные модальные окна

## Особенности реализации

Пример использует высокоуровневый API фреймворка SimpleAPI, который предоставляет удобные функции для создания интерфейсов:

- `createWindow()` - создание окна
- `addLabel()`, `addTextBox()`, `addCheckBox()` и т.д. - добавление виджетов
- `addButton()` - добавление кнопок с обработчиками
- `showMessage()`, `showConfirm()`, `showInput()` - отображение модальных диалогов

## Совместимость

Пример предназначен для работы на платформах Windows и Linux, поддерживаемых фреймворком TextUI.
